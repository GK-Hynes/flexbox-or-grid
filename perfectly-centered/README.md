# Perfectly Centered

Both Flexbox and Grid allow you to center items easily. Grid is marginally more concise.

## Flexbox

```css
.hero {
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
}
```

## CSS Grid

```css
.hero {
  display: grid;
  justify-items: center;
  align-content: center;
}
```
